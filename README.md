# qa-sauce-demo-checkout-testing
# 🛒 QA Testing Project - Sauce Demo Checkout

Este projeto apresenta a documentação e execução de testes manuais para validação do fluxo de compra no Sauce Demo, uma aplicação web demo utilizada para práticas de testes de software.

## 🎯 Objetivo

Validar funcionalidades críticas de um e-commerce, garantindo que o utilizador consiga iniciar sessão, visualizar produtos, adicionar itens ao carrinho, preencher o checkout e finalizar o pedido corretamente.

## 📌 Escopo

O projeto foca nos seguintes fluxos:

- Login
- Listagem de produtos
- Ordenação de produtos
- Adição e remoção de produtos no carrinho
- Validação do carrinho
- Preenchimento do checkout
- Validação de campos obrigatórios
- Finalização do pedido
- Mensagens de erro e confirmação

## 🛠️ Ferramentas utilizadas

- Google Sheets
- GitHub
- Documentação manual de bugs
- Browser: Google Chrome

## 🔍 Tipos de testes realizados

- Testes funcionais
- Testes exploratórios
- Testes de regressão
- Testes positivos e negativos
- Validação de interface
