# Requisitos Funcionais - Sauce Demo Checkout

## RF-001 - Login
O utilizador deve conseguir iniciar sessão com credenciais válidas.

## RF-002 - Mensagem de erro no login
O sistema deve apresentar mensagem de erro ao inserir credenciais inválidas ou campos obrigatórios vazios.

## RF-003 - Visualização de produtos
Após o login, o utilizador deve visualizar a lista de produtos disponíveis.

## RF-004 - Ordenação de produtos
O utilizador deve conseguir ordenar os produtos por nome ou preço.

## RF-005 - Adicionar produto ao carrinho
O utilizador deve conseguir adicionar produtos ao carrinho.

## RF-006 - Remover produto do carrinho
O utilizador deve conseguir remover produtos adicionados ao carrinho.

## RF-007 - Visualizar carrinho
O carrinho deve apresentar os produtos adicionados corretamente.

## RF-008 - Preencher checkout
O utilizador deve conseguir preencher os dados obrigatórios do checkout.

## RF-009 - Validação de campos obrigatórios
O sistema deve bloquear a continuação do checkout se campos obrigatórios não forem preenchidos.

## RF-010 - Finalizar pedido
O utilizador deve conseguir finalizar o pedido e visualizar uma mensagem de confirmação.
